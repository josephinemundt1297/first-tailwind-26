# Modern Tailwind CSS Landing Page

A professional, **mobile-first** landing page built to demonstrate modern **UI/UX design** principles using **Tailwind CSS**. This project focuses on clean layout structures, high-contrast visual hierarchy, and efficient utility-first styling.

## 🚀 Technologies
* **Tailwind CSS** – Utility-first styling for rapid, responsive design.
* **Vite** – Optimized build tool and fast dev server.
* **HTML5** – Semantic structure for better SEO and accessibility.

## ✨ Key Features
* **Fully Responsive** – Seamlessly scales from mobile devices to large desktops.
* **Modern Aesthetic** – Clean typography, balanced whitespace, and professional color palettes.
* **Optimized Assets** – Minimal footprint and fast loading times via Vite's bundling.

## 🛠️ Installation
**1. Clone the repository**

```bash
git clone https://github.com/josephinemundt1297/first-tailwind-26.git
```

**2. Change into the project directory**

```bash
cd first-tailwind-26
```

**3. Install dependencies**

```bash
npm install
```

**4. Start the development server**

```bash
npm run dev
```
